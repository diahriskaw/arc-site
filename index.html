Build on Arc

Quickstarts, tutorials & SDKs

Deploy Solidity contracts with Hardhat, Foundry, Viem, or Ethers. Use Circle Contracts for templates. Pay gas in USDC.

deploy.ts — viem 
import { createWalletClient, http } from 'viem'
import { privateKeyToAccount } from 'viem/accounts'
import { arcTestnet } from './chains'

const account = privateKeyToAccount(process.env.PK)
const client  = createWalletClient({ account, chain: arcTestnet, transport: http() })

// USDC is the gas token on Arc — fees are paid in USDC, not ETH.
const hash = await client.deployContract({ abi, bytecode, args: [] })
console.log('deployed:', hash)
